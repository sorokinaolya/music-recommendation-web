# MoodTune — Music Recommendation Web App

Веб-приложение для музыкальных рекомендаций с учетом настроения пользователя, жанровых предпочтений, истории прослушиваний и избранных треков.

## Структура проекта

- `front/` — frontend на HTML/CSS/JavaScript
- `back/` — FastAPI backend
- `back/app/` — API, авторизация, хранение состояния пользователя, сервис рекомендаций
- `back/artifacts/` — локальная папка для артефактов обученной модели

## Важно

Артефакты обученной модели не включены в репозиторий из-за размера файлов.

Для запуска рекомендательной части нужно локально добавить файлы модели в папку:
